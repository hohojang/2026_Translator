# 🌍 Google 번역기 v1.0 출시!

## ✨ 주요 기능
- 🎨 **현대적 UI**: customtkinter 기반 깔끔한 다크 모드 인터페이스
- 🔄 **자동 언어 감지**: 한국어 ↔ 영어 자동 인식 및 번역
- ⚡ **빠른 번역**: 비동기 처리로 UI가 멈추지 않음
- 📋 **클립보드 복사**: 번역 결과 한 번의 클릭으로 복사
- 🗑️ **초기화 기능**: 입력/결과 영역 쉽게 초기화

## 📦 다운로드 및 설치
1. `Google_Translator_v1.0.zip` 다운로드
2. 압축 해제
3. `Google_Translator.exe` 더블클릭 실행
4. 인터넷 연결 필수 (Google 번역 API 사용)

## 💻 시스템 요구사항
- **OS**: Windows 10 이상
- **메모리**: 최소 200MB
- **인터넷**: 필수 (번역 서비스 사용)

## 🔧 기술 스택
- **언어**: Python 3.13
- **GUI**: customtkinter 5.2.2
- **번역**: deep-translator 1.11.4
- **빌드**: PyInstaller 6.20.0

## 📝 사용 방법
1. **원문 입력**: 상단 텍스트 박스에 번역할 내용 입력
2. **언어 선택**: 한국어→영어 또는 영어→한국어 버튼 클릭
3. **결과 확인**: 하단에 번역 결과 표시
4. **복사**: "📋 결과 복사" 버튼으로 클립보드에 저장
